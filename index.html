ABOUT

Welcome to my portfolio!

You can learn about me and my projects on this website, but please reach out to me for more information.

My name is Lainey Creighton and I am currently a student in the UC Berkley Full Flex Coding Bootcamp. I have been working in hospitality management since 2015. My career in hospitality started in Charlotte, North Carolina. Since then, I have worked my way up and through many different areas of the industry and the country. including Lake Tahoe, Napa Valley, and Seattle. Most recently, I have been working as the Director of Event Technology for events that take place in venues like hotels, resorts and convention centers. I'm very thankful for the experience the hospitality industry has brought me, but I always felt like there was something that would fit me better. I have thought about getting into software development for a couple years. In early 2023, I finally brought those thoughts come to life. I look forward to the opportunities, knowledge, and networking to come. Thank you for reading and I look forward to hearing from you!

Meet Salmon

  • Website
  • GitHub
Description:

This project was made to mimic an on-the-job ticket example of refactoring code to make it follow accessibility standards and honor the Scout Rule - meaning leave the code better than it was found. Adjustments were made to the HTML elements along with the cooresponding CSS selectors to "clean" it up without impacting the code's output. Comments were added to all changes to annotate adjustments. Completing this project teaches that code can still have the same outcome even if many paths are taken, but there is always room for improvement.
